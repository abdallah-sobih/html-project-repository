Year A brief summary of the programming language that was launched
1883 Analytical Engine Algorithm: Ada Lovelace used it in Charles Babbage's analytical engines to calculate Bernoulli numbers, and you are the first computer programming language.
1949 Assembly language: It is used for the first time widely in the automatic calculator to store electronic delay, assembly language is a type of low-level computer programming language that simplifies the language of the device code, and the specific instructions needed to tell the computer what to do.
1952 Autocode: Autocode is a general term for a family of early computer programming languages. The first was developed by Alick Glennie for Computer Mark 1 at the University of Manchester in the United Kingdom, and some consider automatic code to be the first batch computer programming language, which means that it can be translated directly into device code using a program called translator.
1957 Fortran: A computer programming language developed by John Backus for sophisticated scientific, mathematical and statistical work. It is one of the oldest computer programming languages still in use today.
1958 Algol: Implemented by a committee for scientific use, Algol stands for Algorithm Language. Algol served as a starting point in developing languages such as Pascal, C, C ++, and Java.
1959 COBOL: was used by Dr. Grace Murray Hopper as a computer programming language that can run on all brands and types of computers. It is used in ATMs, credit card processing, telephone systems, hospital and government computers, automobile systems and traffic lights. In The Terminator, portions of the COBOL source code were used to display Terminator vision
1959 LISP: Created by John McCarthy from MIT, LISP is still in use. It refers to the LISt processing language. It was originally created for AI research but can be used today in situations where Ruby or Python is used.
1964 BASIC: It was developed by John G. Kemeny and Thomas E. Kurtz at Dartmouth College College so that students who do not have a strong technical or mathematical understanding can use computers, which is an abbreviation for all-purpose instruction code for beginners. A modified version of BASIC. This was to become the first Microsoft product.
1970 Pascal: Developed by Niklaus Wirth, Pascal has been named in honor of the French mathematician, physicist and philosopher Blaise Pascal. It's easy to learn and originally built as a computer programming learning tool. Pascal was the primary language for software use in Apple's early years.
1972 C: Developed by Dennis Richie at Bell Labs, C is the first language to be high-level. The high-level computer programming language is closer to the human language and more removed from the device code. C was created so that an operating system called Unix can be used on many different types of computers. It affected many other languages, including Ruby, C #, Go, Java, JavaScript, Perl, PHP and Python.
1972 SQL: SQL was developed by Donald D. Chamberlain and Raymond F. Boyce at IBM. SQL refers to the structured query language. It is used to view and change information stored in databases. SQL uses query-known query statements to add, remove, or display data.
1978 MATLAB: Developed by Cleve Moler. Matlab stands for Matrix Lab. It is one of the best computer programming languages for writing mathematical programs and is mainly used in mathematics, research and education. It can also be used for 2D and 3D graphics.
1983 C ++: C ++ is an extension of C language developed by Bjarne Stroustrup. It is one of the most widely used languages in the world. C ++ is used in game drivers and high-performance programs such as Adobe Photoshop. Most of the packed programs are still written in C ++.
1987 Perl: Perl was originally developed by Larry Wall in 1987 as a programming language designed to edit text. Its purpose was to facilitate the processing of reports. It is now widely used for many purposes, including Linux system administration, web development, and network programming.
1991 Python: Designed by Guido Van Rossum, Python is easier to read and requires fewer lines of code than many other computer programming languages. It was named after the British comedy group Monty Python. Python.
1991 Visual Basic: Developed by Microsoft, Visual Basic allows programmers to choose and change predefined code fragments in a drag-and-drop method through a graphical user interface (GUI).
1993 R: It was developed by Ross Ihaka and Robert Gentleman at the University of Auckland, New Zealand, to amend the designation R as the first authors. It is used mostly by statisticians and those who do different types of data analysis.
1995 Java: Originally called Oak, Java was developed by Sun Microsystems. It was intended for cable boxes and portable devices but was later improved to be used for presenting information on the World Wide Web. Java is everywhere, from computers to smartphones to parking counters. Three billion devices run Java! .
1995 PHP: Qualified by Rasmus Lerdorf, PHP is mostly used for web development and for running on web servers. It originally symbolized the personal home page, as it was used by Lerdorf to manage his private information online. PHP is now widely used to build websites and blogs. WordPress, a popular website builder, is written using PHP.
1995 Ruby: Ruby was created by Yukihiro Matts Matsumoto, who collects parts of his favorite languages to form a new general-purpose computer programming language that helps you perform many programming tasks. It is common in web application development. Ruby code executes slower, but it allows computer programmers to quickly assemble and run the program.
1995 Javascript: Used in only 10 days by Brendan Eich, this language is mostly used to improve many web browser interactions. Almost every website uses JavaScript.
2000 C #: Developed by Microsoft Combining computing power for C ++ with the simplicity of Visual Basic, C # relies on C ++ which is similar to Java in many aspects. It is used in almost all Microsoft products and mainly used for developing desktop applications.
2003 Scala : Written by Martin Odersky. Scala is a computer programming language that combines functional programming, which is mathematical, with object oriented programming, which is organized around data that govern access to code. Its compatibility with Java makes it useful for Android development.
2003 Groovy: Developed by Java Strachan and Bob McWhirter, Groovy is derived from Java and improves developer productivity because it is easy to learn and concise.
2009 Go: Go was developed by Google to address problems that can occur in large software systems. It is used for computer types and technology is much different today than it was when languages like C ++, Java and Python were introduced, problems arose when huge computer systems became common. The purpose of the go was to improve the work environment for programmers so they went on writing, reading and maintaining software more efficiently.
2014 Swift: Developed by Apple as an alternative to C, C ++ and Objective-C, Swift is supposed to be easier to use and offer less space for errors. It is versatile and can be used for desktop, mobile and cloud services applications.
